# Aha Moments

A collection of insights and small demos—one idea per folder, with code (and sometimes notes) to make the point concrete.

## Structure

Each entry lives in its own folder:

```
YYYYMMDD_<short-title>/
  ├── <short-title>.html    # Interactive or static illustration
  └── README.md             # (optional) Explanation and context
```

- **Folder name**: `YYYYMMDD_<short-title>` — date plus a slug (e.g. `20260129_geometric-interpretation-of-linear-regression`).
- **HTML**: The main illustration (plot, animation, or small app).
- **Markdown**: Optional; use it to spell out the insight, intuition, or links before/after viewing the demo.
